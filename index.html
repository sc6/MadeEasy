Text is anything that can be written. Since markup languages markup text, markup languages can understand anything that can be written!
All the text in between the opening pointy bracket < and the first space is called the element name. The element name is underlined in the following:
<elementname>
<thisisanelementname thisisnotanelementname>
<stilltheelementname someattribute=\"value\">

Markup languages can convert elements into things on the screen.

If you want to put things inside an element, you would use opening and closing tags.
Closing tags always start with a slash / after the beginning pointy bracket <.
The opening tag is green and the closing tag is blue in the following:
<opening-tag> stuff you want to put in your element <maybeanelement/> </closing-tag>
